SenseEd

PROJECT SUMMARY
SenseEd delivers an immersive, neon-gradient dashboard experience that helps teachers monitor classroom engagement in real time. With clean login flows, intuitive charts, and emotion-driven insights, the UI is designed for clarity, speed, and smarter decision-making. Every screen—from analytics to attendance—uses modern visual hierarchy to make classroom intelligence effortless and visually compelling.
PROBLEM STATEMENT
Classrooms lack real-time insights into student attention, emotion, and learning patterns. SenseEd aims to solve this by providing a beautiful, accessible dashboard that makes complex analytics easy to understand at a glance.
MY ROLES
  • UI/UX Designer
  • Frontend Designer (Figma Prototype)
  • Interaction Design
  • Visual Branding & Logo Design
  • Conceptual Architecture
GOALS OF UI
  • Create a clean, immersive, futuristic interface using neon gradients.
  • Visualize emotion + engagement data clearly using graphs and charts.
  • Build an inclusive experience for teachers and students.
  • Maintain visual consistency across login, dashboard, profile, calendar, and FAQs screens.
TARGET AUDIENCE
  • School teachers
  • Students
  • Admins/Institutions
  • Special education departments
  • Research
    • Studied accessibility needs (deaf, blind, low-engagement learners).
    • Analysed gaps in existing educational tools (Google Classroom, Edmodo, etc.).
  • Wireframing
    • Created low-fidelity layouts for dashboards and login screens.
    • Prioritized simplicity and clarity.
  • High-Fidelity UI Design
    • Designed neon purple-pink gradients for a modern feel.
    • Developed charts, cards, iconography, and navigation.
    • Ensured spacing, contrast, and readability.
  • Prototyping
    • Interactive dashboard transitions.
    • Smooth navigation between sections.
VISUAL IDENTITY
  • Color Palette: Purple, pink, deep blue gradients
  • Typography: Clean, modern sans-serif
  • Design Language: Soft edges, glow effects, minimal icons
  • Branding: Custom logo combining a book + neural connections
UI CHALLANGES & SOLUTIONS
  • Challange:
    • Balancing bright gradients with text readability.
    • Solution: Strategic use of dark cards + opacity layers.
  • Challange:
    • Displaying complex analytics without overwhelming the user.
    • Solution: Hierarchical layout and focused visual grouping.
  • Challange:
    • Handling multiple user roles.
    • Solution: Unified navigation with recognizable icons.
